# 貢獻者公約

## 我們的承諾

為了營造開放且友善的環境，我們作為貢獻者和維護者承諾：不論年齡、體型、身心障礙、種族、性別特徵、性別認同與表達、經驗水平、國籍、個人外貌、種族、宗教或性取向，都為我們的社群成員提供不受騷擾的體驗。

## 我們的標準

有助於創造正面環境的行為包括：

* 使用友善和包容的語言
* 尊重不同的觀點和經驗
* 優雅地接受建設性批評
* 關注對社群最有利的事情
* 對其他社群成員展現同理心

不可接受的參與者行為包括：

* 使用帶有性暗示的語言或影像，以及不受歡迎的性關注或挑逗
* 挑釁、侮辱/貶損的評論，以及個人或政治攻擊
* 公開或私下的騷擾行為
* 未經明確許可，公開他人的私人資訊，例如住址或電子郵件地址
* 其他可以被合理地認定為不適當或違反專業環境的行為

## 我們的責任

專案維護者有責任為可接受行為的標準做出澄清，並對任何不可接受的行為採取適當且公平的糾正措施。

專案維護者有權利和責任移除、編輯或拒絕不符合本行為準則的評論、提交、程式碼、維基編輯、議題和其他貢獻，或暫時或永久禁止任何貢獻者進行他們認為不適當、威脅、冒犯或有害的其他行為。

## 適用範圍

當個人代表專案或其社群時，本行為準則適用於專案空間和公共空間。代表專案或社群的例子包括使用官方專案電子郵件地址、透過官方社群媒體帳號發文，或在線上或線下活動中擔任指定代表。專案維護者可進一步定義和澄清代表專案的具體情況。

## 執法

如遇虐待、騷擾或其他不可接受的行為，請透過 [INSERT CONTACT METHOD] 聯絡專案團隊。所有投訴將被審查和調查，並在情況需要下做出適當回應。專案團隊有義務對事件舉報者的隱私保密。具體執法政策可能會單獨發佈。

未真誠遵守或執行本行為準則的專案維護者，可能會面臨由專案領導層其他成員決定的暫時或永久性後果。

## 聯絡資訊

對於行為準則相關問題，請聯絡：

* **主要聯絡人**：專案維護團隊
* **電子郵件**：conduct@example.com（請替換為實際聯絡信箱）

我們承諾在 **48 小時內** 回應所有行為準則相關的查詢。

## 執法流程

### 1. 報告
收到行為準則違規報告後，維護團隊將：
- **24 小時內**：確認收到報告
- **3 個工作日內**：進行初步調查
- **7 天內**：完成調查並提出處理方案

### 2. 調查
調查過程將：
- 保護舉報者隱私（除非法律要求）
- 聽取所有相關方的說法
- 以公正、客觀的態度進行

### 3. 處置
可能的處置措施包括：
- **警告**：書面警告，要求停止不當行為
- **暫時禁止**：在一定期間內禁止參與專案活動
- **永久禁止**：永久禁止參與專案活動

### 4. 上訴
對處置決定不服者，可在 **14 天內** 提出上訴。上訴將由未參與原調查的維護者審理。
